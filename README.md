## 简介
`magnetW`基于[magnetX](https://github.com/youusername/magnetX)的规则原理，将各个磁力站的搜索结果统一格式化

## 安装
从[官网](https://app.bt-mao.com)或者[Github Releases](https://github.com/tiseheaini/magnetX/releases)下载对应平台
